Struct ClientFork

pub struct ClientFork {
    pub storage: Arc<RwLock<ForkedStorage>>,
    pub config: Arc<RwLock<ClientForkConfig>>,
    pub database: Arc<RwLock<Box<dyn Db>>>,
}
Expand description

Represents a fork of a remote client

This type contains a subset of the EthApi functions but will exclusively fetch the requested data from the remote client, if it wasn’t already fetched.

Fields§

§storage: Arc<RwLock<ForkedStorage>>

Contains the cached data

§config: Arc<RwLock<ClientForkConfig>>

contains the info how the fork is configured

§database: Arc<RwLock<Box<dyn Db>>>

This also holds a handle to the underlying database
