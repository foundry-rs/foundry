# futures-io

The `AsyncRead`, `AsyncWrite`, `AsyncSeek`, and `AsyncBufRead` traits for the futures-rs library.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
futures-io = "0.3"
```

The current `futures-io` requires Rust 1.36 or later.
