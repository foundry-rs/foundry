pub struct CreateOutcome {
    pub result: InterpreterResult,
    pub address: Option<Address>,
}
Expand description

Represents the outcome of a create operation in an interpreter.

This struct holds the result of the operation along with an optional address. It provides methods to determine the next action based on the result of the operation.
